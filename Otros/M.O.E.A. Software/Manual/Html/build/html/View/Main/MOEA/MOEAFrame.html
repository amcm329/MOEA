MOEAFrame (clase)¶

class MOEAFrame(parent, features)¶

Bases: Tkinter.Frame

Unifica los Frames AlgorithmFrame y SharingFunctionFrame, la razón de ésto es para facilitar el acomodo de componentes de manera individual, para así garantizar un acceso asequible a la información.

Parameters:
  • parent (Tkinter.Frame) – Frame padre al que pertenece.
  • features (Dictionary) – Conjunto de técnicas con sus respectivos parámetros para que se puedan cargar automáticamente en este Frame (véase Controller/XMLParser.py).
Returns:

Tkinter.Frame
Return type:

Instance
