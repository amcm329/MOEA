GeneticOperatorFrame (clase)¶

class GeneticOperatorFrame(parent, features)¶

Bases: Tkinter.Frame

Reúne y controla las clases SelectionFrame, CrossoverFrame y MutationFrame con la finalidad de colocar los elementos gráficos apropiadamente y agilizar el intercambio de información con el usuario.

Parameters:
  • parent (Tkinter.Frame) – Frame padre al que pertenece.
  • features (Dictionary) – Conjunto de técnicas con sus respectivos parámetros para que se puedan cargar automáticamente en este Frame (véase Controller/XMLParser.py).
Returns:

Tkinter.Frame
Return type:

Instance
