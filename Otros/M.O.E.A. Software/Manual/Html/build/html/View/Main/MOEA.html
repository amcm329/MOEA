View.Main.MOEA.AlgorithmFrame module¶

class View.Main.MOEA.AlgorithmFrame.AlgorithmFrame(parent, name, features)¶

Bases: Tkinter.Frame

Esta clase proporciona una base gráfica para que el usuario pueda seleccionar técnicas con sus parámetros correspondientes (si es que tienen) referentes a los MOEAs (Multi Objective Evolutionary Algorithms).

Parameters:
  • parent (Tkinter.Frame) – Frame padre al que pertenece.
  • name (String) – Identificador (único) que tendrá el Frame.
  • features (Dictionary) – Conjunto de técnicas con sus respectivos parámetros para que se puedan cargar automáticamente en este Frame (véase Controller/XMLParser.py).

class View.Main.MOEA.MOEAFrame.MOEAFrame(parent, features)¶

class View.Main.MOEA.SharingFunctionFrame.SharingFunctionFrame(parent, name, features)¶

Esta clase proporciona una base gráfica para que el usuario pueda seleccionar métosos con sus respectivos parámetros (si es que tienen) referentes a los métodos de Sharing Function.
