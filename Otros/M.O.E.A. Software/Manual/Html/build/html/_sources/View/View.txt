View (sección)
==============


La capa View **(ó Vista)** contiene todos los elementos que serán 
alusivos a la interfaz gráfica. De acuerdo al modelo MVC **(Model-View-Controller)**,
opera exclusivamente con la capa Controller **(ó Controlador)**.
